1. twosum

Given an array of integers, return indices of the two numbers such that they add up to a specific target. You may assume that each input would have exactly one solution.

Example:

Given nums = [2, 7, 11, 15], target = 9, Because nums[0] + nums[1] = 2 + 7 = 9,

return [0, 1].

UPDATE (2016/2/13): The return format had been changed to zero-based indices. Please read the above updated description carefully.

  • 第一次解：使用兩個index, i, j, 跑兩個for迴圈，兩兩相比後可以得出解答 O(n*n)
static int[] twoSum(int[] nums, int target) {
    for(int i=0; i<nums.length; i++){
        for(int j=0; j<nums.length;j++){
            if(i!=j&&(nums[i]+nums[j]==target)){
                return new int[]{i+1, j+1};
            }
        }
    }
    return new int[]{999,999};
}
  • 第二次解：two pass, 先掃一遍，把value跟index記在map, 第二次掃的時候，除了自己，使用map搜尋出剩餘的數 : 要搜尋的數 = target - 目前的數
static int[] twoSum1(int[] nums, int target) {
    Map<Integer, Integer> table = new HashMap<Integer, Integer>();
    for(int i=0; i<nums.length; i++){
        table.put(nums[i], i);
    }
    for(int i=0; i<nums.length;i++){
        int search = target - nums[i];
        Integer idx = table.get(search);
        if(idx==null){
            continue;
        }else{
            if(idx==i){
                continue;
            }else{
                return new int[]{idx, i};
            }
        }
    }
    return new int[]{999,999};
}
