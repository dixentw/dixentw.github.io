242. Valid Anagram

Given two strings s and t, write a function to determine if t is an anagram of s.

For example, s = “anagram”, t = “nagaram”, return true.
s = “rat”, t = “car”, return false.

Note: You may assume the string contains only lowercase alphabets.

Follow up: What if the inputs contain unicode characters? How would you adapt your solution to such case?

  • 第一次想法：使用List, 把s的characters都掃進去，然後針對t的character, 一個一個去消掉Map裡面的character，最後map的size為空就是anagram。
public static boolean isAnagram(String s, String t) {
    List<Character> allChar = new ArrayList<Character>();
    for(int i=0; i<s.length();i++){
        allChar.add(s.charAt(i));
    }
    for(int j=0; j<t.length();j++){
    	Character c = new Character(t.charAt(j));
    	if(allChar.contains(c)){
    		allChar.remove(c);
    	}else{
    		return false;
    	}
    }
    return allChar.size() == 0;
}
  • 別人學來的：把s, t sort過一遍，對每個s, 去找t同一個index的字元，如果有不是，就不是anagram。
public static boolean isAnagram2(String s, String t){
	char[] scs = s.replaceAll("\\s", "").toCharArray();
	char[] tcs = t.replaceAll("\\s", "").toCharArray();
	Arrays.sort(scs);
	Arrays.sort(tcs);
	if(scs.length!=tcs.length){
		return false;
	}else{
		for(int i=0; i<scs.length; i++){
			if(scs[i]!=tcs[i]){
				return false;
			}
		}
		return true;
	}
}
